# __Introducing Givify__
**Givify** is a powerful nodejs module that allows you to easily create giveaways. It is designed to be simple and easy to use, with a focus on stability and more features.

## __Features__
- Easy To Use
- Buttons
- Support for all databases (default is json)
- Super powerful
- Very Customizable
