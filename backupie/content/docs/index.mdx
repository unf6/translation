---
title: Introducing Backupie
description: Backupie is a powerful nodejs module that allows you to easily manage discord server backups. It is designed to be simple and easy to use, with a focus on stability and more features.
---

## __Features__
- Easy To Use
- Unlimited backups
- Backup creation takes less than 10 seconds
- Even restores messages with webhooks
- Restores everything that is possible to restore (channels, roles, permissions, bans, emojis, name, icon, and more!)
- Super powerful
